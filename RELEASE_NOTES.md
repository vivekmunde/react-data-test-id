# Release Notes

## v1.0.0

Initial public release of `react-data-test-id`.

### Highlights:

- Context-based `data-test-id` scoping with `DataTestIdRoot` and `DataTestId`.
- Configurable behavior for `enabled`, `pathSeparator`, `dataAttributeName`, `spaceReplacement`, and `caseTransform`.
- Dedicated provider, hooks, and internal building blocks for composability.
- Comprehensive documentation and examples.
- 100% test coverage enforcement in CI and pre-commit.
